# Android-CurrencyConverter
Android: CurrencyConverter using Broadcast Receiver.

1. In the Currency Converter Application, enter the dollar amount to be converted and choose the currency type to be converted to.

![1](https://user-images.githubusercontent.com/42703827/54081725-27c5ba80-42be-11e9-9d30-82d9a2a71818.png)

2. In the Currency Conversion Exchange, click on Apply to see the toast message with the converted value.

![3](https://user-images.githubusercontent.com/42703827/54081729-3d3ae480-42be-11e9-8959-3b8634f69341.png)

3. Back to Currency Converter Application, verify the converted amount message being displayed below:

![4](https://user-images.githubusercontent.com/42703827/54081730-44fa8900-42be-11e9-820d-6da754dd29b2.png)
